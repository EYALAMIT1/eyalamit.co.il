# איך לבדוק ש-Google Site Kit עובד בסביבה מקומית

## בדיקות בסיסיות

### 1. בדיקה שהפלאגין פעיל
הפלאגין פעיל ומותקן בגרסה 1.165.0 ✅

### 2. בדיקה דרך דף Admin

#### שלב 1: התחברות ל-Admin
1. פתח את הדפדפן וגש ל: `http://localhost:8080/wp-admin`
2. התחבר עם פרטי המשתמש שלך

#### שלב 2: בדיקה שהפלאגין מופיע
1. לך ל: **Plugins > Installed Plugins**
2. חפש את **"Site Kit by Google"**
3. ודא שהוא מופיע כ-**Active** (פעיל)

#### שלב 3: בדיקה בדף Site Kit
1. בדף Admin, לחץ על **"Site Kit"** בתפריט השמאלי
2. אם אתה רואה את מסך ההתקנה/הגדרה של Site Kit - זה אומר שהפלאגין עובד! ✅
3. אם אתה רואה שגיאה - זה אומר שיש בעיה

### 3. בדיקת REST API (נדרש ל-Site Kit)

#### דרך 1: בדיקה בדפדפן
פתח בדפדפן:
```
http://localhost:8080/wp-json/wp/v2/
```

אם אתה רואה JSON עם מידע על WordPress - REST API עובד ✅

#### דרך 2: דרך WP-CLI
```bash
docker exec newwebsiteainov2025-wordpress-1 wp option get home --allow-root --path=/var/www/html
```

### 4. בדיקת שגיאות PHP

#### דרך 1: בדיקת לוגים
```bash
docker logs newwebsiteainov2025-wordpress-1 | grep -i "site-kit\|fatal\|error" | tail -20
```

#### דרך 2: בדיקה ב-wp-config.php
ודא ש-`WP_DEBUG` מוגדר ל-`false` (כדי לא להציג שגיאות למשתמשים):
```php
define("WP_DEBUG", false);
```

### 5. בדיקה שהפלאגין יכול להתחבר (אופציונלי)

**הערה חשובה:** בסביבה מקומית, Site Kit לא יכול להתחבר בפועל לחשבון Google כי:
- האתר לא נגיש פומבית (localhost)
- Google דורש שהאתר יהיה נגיש פומבית

**אבל** - אתה יכול לבדוק שהפלאגין מוכן להתחברות:
1. לך ל: **Site Kit > Settings**
2. אם אתה רואה כפתור "Sign in with Google" או מסך התחברות - הפלאגין עובד! ✅
3. ההתחברות בפועל תעבוד רק באתר ייצור (לא ב-localhost)

## מה זה אומר שהפלאגין "עובד"?

### ✅ הפלאגין עובד אם:
- הפלאגין פעיל ב-Admin
- אתה יכול לגשת לדף Site Kit ב-Admin
- אין שגיאות PHP בלוגים
- REST API עובד

### ❌ הפלאגין לא עובד אם:
- אתה רואה שגיאות PHP
- הפלאגין לא מופיע ב-Admin
- אתה לא יכול לגשת לדף Site Kit
- REST API לא עובד

## הערות חשובות

1. **בסביבה מקומית** - Site Kit לא יכול להתחבר בפועל לחשבון Google, אבל הפלאגין עצמו יכול לעבוד
2. **באתר ייצור** - אחרי שתעתיק את העדכונים לייצור, תוכל להתחבר לחשבון Google
3. **העדכון הצליח** - הפלאגין עודכן מ-1.43.0 ל-1.165.0, מה שאומר שהוא תואם ל-WordPress 6.8.3

## בדיקה מהירה (30 שניות)

1. פתח: `http://localhost:8080/wp-admin`
2. לחץ על **"Site Kit"** בתפריט השמאלי
3. אם אתה רואה את מסך Site Kit (גם אם הוא מבקש התחברות) - הכל עובד! ✅

## אם יש בעיות

אם אתה רואה שגיאות:
1. בדוק את לוגי Docker: `docker logs newwebsiteainov2025-wordpress-1`
2. בדוק את לוגי PHP (אם יש)
3. ודא ש-REST API עובד
4. ודא שהפלאגין פעיל
