1. פתח PowerShell או CMD
2. נווט לתיקייה 1:
   ```cmd

3. עצור את הקונטיינרים:

   ```

**או דרך Docker Desktop:**

- מצא את הקונטיינרים: `newwebsiteainov2025-db-1`, `newwebsiteainov2025-wordpress-1`, וכו'
