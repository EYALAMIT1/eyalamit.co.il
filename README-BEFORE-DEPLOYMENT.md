1. פתח Admin: `http://localhost:8080/wp-admin`
2. עבור ל: **Dashboard → Updates**
3. בדוק אם יש עדכונים זמינים
4. **חשוב במיוחד:**

   - RevSlider - 🟡 מומלץ
