# מדריך מודרניזציה - אתר WordPress

## שלבים לביצוע

### שלב 1: גיבוי מלא ✅
**בוצע!** יש לך גיבוי בתיקייה `backups\backup_2025-11-24_14-00-14\`

### שלב 2: הפעלת Docker עם הגרסאות החדשות
לחץ פעמיים על: **`START-DOCKER.bat`**

זה יפתח את:
- WordPress עם PHP 8.2
- MariaDB 10.11
- Nginx עם הגדרות SEO
- phpMyAdmin

**אחרי שהקונטיינרים רצים**, בדוק שהאתר עובד: http://localhost:8080

### שלב 3: שדרוג WordPress Core
לחץ פעמיים על: **`UPGRADE-WORDPRESS.bat`**

זה ישדרג את ליבת WordPress לגרסה 6.8.3.

### שלב 4: בדיקת האתר
לחץ פעמיים על: **`CHECK-SITE.bat`**

זה יציג:
- סטטוס קונטיינרים
- גרסת WordPress
- רשימת תוספים פעילים
- תבניות פעילות

### שלב 5: שדרוג תוספים (לאחר מכן)
לאחר שתבדוק שהאתר עובד, נמשיך לשדרוג התוספים לפי התכנית.

---

## קבצים חשובים

- **`backup-site.bat`** - גיבוי מלא (DB + קבצים)
- **`START-DOCKER.bat`** - הפעלת Docker
- **`UPGRADE-WORDPRESS.bat`** - שדרוג WordPress
- **`CHECK-SITE.bat`** - בדיקת סטטוס

---

## הערות חשובות

1. **לפני כל שינוי** - הרץ `backup-site.bat`
2. **אם משהו לא עובד** - בדוק ש-Docker Desktop רץ
3. **אם יש שגיאות** - שלח את ההודעות המדויקות
