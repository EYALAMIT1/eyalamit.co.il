## 📋 מה צריך לעשות עכשיו - צעד אחר צעד

### שלב 1: עדכון GitHub (2 דקות)

**להריץ ידנית:**
1. פתח את תיקיית הפרויקט ב-Windows Explorer
2. לחץ לחיצה כפולה על: **`ADD-AND-PUSH.bat`**
3. הסקריפט יעדכן את GitHub עם כל הקבצים החדשים

**מה זה יעשה:**
- יוסיף את כל הקבצים החדשים ל-Git
- יבצע commit עם הודעה מפורטת
- יבצע push ל-GitHub

**אם תתבקש credentials:**
- Username: השם שלך ב-GitHub
- Password: Personal Access Token (לא סיסמה!)

### שלב 2: גיבוי ייצור (חובה!) (15-30 דקות)

**⚠️ אל תדלג על זה! זה הכי חשוב!**

#### 2.1 גיבוי בסיס נתונים

**דרך phpMyAdmin:**
1. פתח phpMyAdmin (דרך cPanel או ישירות)
2. בחר את בסיס הנתונים של האתר (`deveyala_uprdb` או שם דומה)
3. לחץ על **"Export"** (יצוא)
4. בחר **"Quick"** או **"Custom"**
5. לחץ על **"Go"**
6. שמור את הקובץ בשם: `production_backup_DB_2025-11-16.sql`

**איפה לשמור:**
- למחשב המקומי
- בתיקייה בטוחה (לא על השרת!)
- מומלץ: `C:\Users\USER\Pictures\סטודיו נשימה מעגלית\backups\production\`

#### 2.2 גיבוי קבצים

**דרך cPanel File Manager (הכי קל):**
1. פתח cPanel → File Manager
2. עבור לתיקיית האתר (בדרך כלל `public_html` או שם האתר)
3. בחר את כל הקבצים → לחץ לחיצה ימנית
4. בחר **"Compress"** → בחר **"ZIP"**
5. המתן עד שהדחיסה מסתיימת
6. לחץ לחיצה ימנית על הקובץ ZIP → **"Download"**

**איפה לשמור:**
- למחשב המקומי
- בתיקייה בטוחה (לא על השרת!)

**או דרך FTP:**
1. התחבר ל-FTP (FileZilla או כלי אחר)
2. עבור לתיקיית האתר
3. הורד את כל התיקייה למחשב המקומי
